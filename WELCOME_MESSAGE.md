# 🌊 Auditor de Seguridad Ambiental: Cuenca de Tula

## ¡Bienvenido al sistema de análisis normativo y académico!

Esta herramienta está diseñada para sustentar nuestra **Propuesta de Actualización Ambiental** con el máximo rigor técnico.

---

### 📋 Instrucciones de Uso

#### 1️⃣ **Carga de Archivos**
Sube hasta 5 documentos (Artículos, NOMs, Leyes o Informes). El sistema identificará automáticamente el **título real** y generará la ficha técnica en **APA 7.ª edición**.

#### 2️⃣ **Consulta Rápida**
Puedes chatear con los documentos para resolver dudas específicas sobre:
- Contaminantes
- Límites permisibles
- Hallazgos clave
- Normativa aplicable

#### 3️⃣ **Informe Maestro**
Para generar el **dictamen técnico de 2,500 palabras** con visualizaciones interpretadas y la **Matriz de Intersección**, escribe el comando:

```
/Informe_Profundo
```

#### 4️⃣ **Exportación**
Una vez generado el informe, utiliza el botón de **"Exportar a PDF"** para obtener el documento oficial.

---

### ⚠️ Nota Importante

Si un documento no tiene autor personal, el sistema aplicará **autoría corporativa o institucional** automáticamente según norma APA 7.ª edición.
